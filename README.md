# AKYL — Платформа профессионального управления МЖД

AKYL — это веб-платформа для популяризации и внедрения методологии профессионального управления многоквартирными жилыми домами.

Проект объединяет:
- методологию управления МЖД;
- цифровые инструменты анализа;
- библиотеку знаний;
- визуальный атлас управления;
- Центр компетенции AKYL.

## Основная идея

AKYL показывает многоквартирный дом не просто как объект обслуживания, а как управляемую систему, где связаны:

- процессы;
- финансы;
- участники;
- данные;
- KPI;
- отчётность;
- цифровой контроль.

Цель платформы — сделать управление домами более прозрачным, измеримым и профессиональным.

## Основные разделы сайта

## Ключевые модули

### Индекс эффективности управления МЖД

IEU — интегральный показатель от 0 до 100, который отражает качество управления домом.

Он учитывает:

- финансы;
- эксплуатацию;
- сервис;
- подрядчиков;
- управление.

Индекс помогает оценивать уровень управления, находить слабые зоны и сравнивать дома между собой.

### Анализ бюджета

Инструмент для оценки бюджета дома, план-факт анализа и выявления отклонений.

### Чек-листы

Практические чек-листы для самопроверки качества управления, аудита процессов и подготовки к внедрению системы.

### Управленческий отчёт

Модуль для формирования отчёта по ключевым показателям управления МЖД.
